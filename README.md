# ObVim

An Integrated Writing Evironment [^1]

## Features

- dynamic workspaces for editing any markdown
- colorscheme
  - proper checkbox hl

## Keymaps

- [x] \f + \r for footnotes
- [x] outline motion enhancement
- [ ] <tab> / <s-tab> for indenting
- [ ] style mappings for bold, italic, comment ...
